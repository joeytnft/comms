# Assets Directory

## Images
Place your app icons and images here:
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1284x2778 for iPhone 14 Pro Max)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)

## Fonts
Custom fonts go in the `fonts/` directory.

## Sounds
Alert tones and PTT beeps go in the `sounds/` directory.
